SAAFMomentScattering

buildconstruction:Undocumented Class

The SAAFMomentScattering has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Computes the scattering term for the current group of the SAAF discrete ordinates neutron transport equation. The weak form is given by . The group flux moments must be provided to this kernel. This kernel should not be exposed to the user, instead being enabled through a transport action.

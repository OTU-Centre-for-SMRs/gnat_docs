SpecificActivity

buildconstruction:Undocumented Class

The SpecificActivity has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

An auxkernel for the purpose of computing the specific activity of a radionuclide concentration for postprocessing.

Overview

Example Input File Syntax

    Description:The decay constant of the radionuclide scalar field.

    Controllable:No

    Description:Whether the mass fraction is finite element or not.

    Description:The name of the variable that this object applies to

Setting this to false will allow contribution to a single element's elemental value(s) from multiple boundary sides on the same element (example: when the restricted boundary exists on two or more sides of an element, such as at a corner of a mesh

    Default:True

    C++ Type:bool

    Controllable:No

    Description:Whether to check for multiple element sides on the boundary in the case of a boundary restricted, element aux variable.

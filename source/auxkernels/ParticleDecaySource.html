ParticleDecaySource

buildconstruction:Undocumented Class

The ParticleDecaySource has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Auxkernel which computes a particle source from the radioactive decay of a nuclide concentration field. Useful for coupled radiation / mass transport for radiological consequence assessment.
